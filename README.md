# CS300-Portfolio
CS 300 portfolio projects including data structures and algorithms work

Reflection

In this course, the main problem I was solving was how to efficiently store, organize, and retrieve data using different data structures. I worked on building programs that could read data from a file, store it using structures like vectors, hash tables, and binary search trees, and then search or display that data.

I approached the problem by learning how each data structure works and when it should be used. Data structures are important because they affect how fast and efficient a program is. For example, I chose a binary search tree for Project Two because it allows for efficient searching and automatically keeps data sorted.

One of the challenges I faced was working with file input and debugging errors related to reading data correctly. I also ran into issues with CSV formatting and parser errors. I solved these problems by simplifying my approach and using basic file input methods like ifstream and stringstream.

This project helped me improve how I design programs. Writing pseudocode first helped me plan the structure before coding, which made the implementation easier. I also improved my ability to write readable and maintainable code by using clear variable names, comments, and organized sections.
